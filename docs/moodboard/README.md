# Moodboard

**What goes here:** feeling and direction — not necessarily UI at all.

Drop images that answer the question: *"what should this feel like?"*

Examples:
- Color palettes or swatches
- Typography specimens
- Brand identity references
- Abstract photography with the right mood
- A poster, album cover, or editorial layout you like the tone of
- Texture, grain, or material references

The moodboard sets the **emotional register** of the project — the vibe, the personality, the aesthetic direction. Claude reads these to understand *tone*, not layout.

Use any format: `.png`, `.jpg`, `.webp`

---

> Run `/generate-visual-direction` after adding images. Claude synthesises both `moodboard/` and `screenshots/` into `docs/visual-direction.md` — a compact design brief it can reference every session without re-reading your images.
